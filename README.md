## 项目简介

TXCWeb 是一个简洁优雅的个人主页项目,采用了现代化的设计风格和响应式布局。适用于个人博客、作品集展示或个人名片网站。

## 功能特性

- 🎨 现代化UI设计
- 📱 完全响应式布局
- 🚀 快速加载
- ⚡ 轻量级实现
- 🔗 友情链接管理
- ⏰ 实时日期时间显示
- 📊 网站运行时间统计
- 🎯 多站点聚合展示

## 技术栈

- **后端**: PHP
- **前端**: HTML5 + CSS3 + JavaScript
- **图标**: SVG Iconfont
- **设计**: 自定义CSS框架

## 安装使用

1. 克隆本项目到本地:
```bash
git clone https://github.com/2768651338/TXCWeb.git
```

2. 将项目文件部署到PHP8.1环境(如Apache/Nginx)，其它PHP版本可能存在兼容性问题

3. 访问网站主页

## 自定义配置

在 `index.php` 文件中,你可以修改以下内容:

- QQ链接: 修改 `$qq_link` 变量
- 个人信息: 修改姓名、标签等内容
- 链接地址: 修改旗下站点和友情链接
